If you want to build complete eCAL setup for windows then simply call:

```bash
win_make_all.bat
```

If you want to cmake / build / setup eCAL step by step do the following steps:

```bash
win_make_cmake.bat
win_make_build.bat
win_make_setup.bat
```

If you want to create a python wheel call:

```bash
win_make_python.bat
```
