=======================
Development Setup
=======================

Depending on the programming language you use to develop eCAL applications, the setup required to build and run the samples differs.
In general, you will need the language bindings of choice, and language-specific setup will be described in the following:

* C/C++: :doc:`setup/cpp`
* Python: :doc:`setup/python`
