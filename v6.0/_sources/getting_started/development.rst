=================
Development (C++)
=================

You have learned a lot about basic eCAL communication and the eCAL Tools that it comes with.
Now it is time to write your own eCAL program!

.. seealso::
   We are going to start with C++, as this is the language eCAL is written in and most users are going to use for developing performance-critical applications.
   eCAL however also offers a great python integration, so if you are more familiar with python, you may also want to take a look at the :ref:`getting_started_development_python` chapter first!
